Hide My Tabs!

Privacy controls and documentation

Appearance

Hidden tabs are replaced with a neutral "New Tab" style page that includes a privacy indicator. This prevents others from seeing your tab content while maintaining a normal browser appearance.

Privacy

100% Local Storage: All data is stored locally in your browser. No information is ever sent to external servers.
What Gets Stored
  • Original tab URLs
  • Tab titles
  • Favicon URLs
  • Saved session configurations
  • Usage statistics
Data Removal

All stored data for a tab is automatically deleted when you unhide it or close the tab.

How to Use

Hiding Tabs

Single Tab (Current):

  1. Click the extension icon to open the popup
  2. In the "Current Tab" section, click "Hide This Tab"
  3. The tab will be replaced with a mask page

Multiple Tabs (Sessions):

  1. Click the extension icon to open the popup
  2. In the "Tab Sessions" section, select tabs using checkboxes
  3. Click "Hide Selected" to hide all selected tabs

Using Context Menu:

  1. Select one or more tabs (Hold Ctrl or Cmd to select multiple)
  2. Right-click on a selected tab
  3. Select "Hide this tab" or "Hide other tabs"
Showing Tabs

Single Tab:

  • Open popup on the hidden tab and click "Show This Tab"
  • Or right-click anywhere and select "Unhide all tabs"

Multiple Tabs:

  • Use the popup's "Unhide All" button
  • Or right-click anywhere and select "Unhide all tabs"
Saving Sessions
  1. Select tabs you want to save as a session
  2. Click "Save as Session"
  3. Enter a name for the session
  4. Click "Save"

Saved sessions appear in the popup and can be loaded with one click to reselect the same tabs.

Tip: In Firefox, you can right-click directly on the tab strip for quick access to hide options.

Privacy & Security

100% Local Storage: All data is stored locally in your browser. No URLs are ever sent to external servers.
What Gets Stored
  • Original tab URL
  • Tab title
  • Favicon URL (if available)
  • Saved session configurations
Data Removal

When you unhide tabs or close hidden tabs, all stored data is automatically removed.

About Hide My Tabs!

Hide My Tabs! helps you maintain privacy by quickly hiding sensitive browser tabs.

Version

1.0.0 (Manifest V3)

Privacy

This extension works entirely locally in your browser. No data is collected or sent to external servers.

Key Features
  • Hide single or multiple tabs with content masking
  • Save and load tab selections as sessions
  • Context menu integration for quick actions
  • Cross-browser support (Chrome, Edge, Firefox)
  • Manifest V3 compliant
  • 100% local data storage
